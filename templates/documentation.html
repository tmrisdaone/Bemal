Getting Started

API Base URL

https://api.neonoutreach.com/v1

Quick Start

curl -X POST \
  https://api.neonoutreach.com/v1/campaigns \
  -H 'Authorization: Bearer YOUR_API_KEY' \
  -H 'Content-Type: application/json' \
  -d '{
    "name": "My Campaign",
    "template_id": "temp_1234"
  }'

Authentication

All API requests require an API key passed in the Authorization header:

Authorization: Bearer YOUR_API_KEY

Generate your API key from the Developer Settings in your account dashboard.
